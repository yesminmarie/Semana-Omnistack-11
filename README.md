# Semana Omnistack 11
O projeto “Be The Hero” foi desenvolvido durante a Semana Omnistack 11.0, um evento da Rocketseat, e tem como objetivo conectar ONGs a pessoas que desejam contribuir monetariamente com projetos sociais.
## Tecnologias usadas
- Node.js
- ReactJS
- React Native
- Expo
